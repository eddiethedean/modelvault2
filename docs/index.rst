ModelVault Documentation
========================

**Your models are your source of truth. ModelVault keeps them that way.**

ModelVault is a model integrity layer for Python. It persists ordinary Pydantic models
to trusted databases (SQLite in v0.17) with validation on read and write, contract
registry metadata, and drift detection.
